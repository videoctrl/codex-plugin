Run verification before approval.

Use:
- `score_bookmarkability`
- `run_avoid_slop_check`
- `run_viral_postmortem`
- `score_ad_readiness`
- `validate_for_platform`

Return clear results:
- total score
- strongest row
- weakest row
- verdict
- specific fix

Do not approve content below the threshold. Send it back to the packet, script, or timeline section that caused the weakness.
