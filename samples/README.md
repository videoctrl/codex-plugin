# VideoControl Samples

Run this from the repo root to create three small local projects with generated media, previews, contact sheets, exports, and content folders:

```bash
pnpm create:samples
```

The samples are:

- `product-demo`: a short landscape product walkthrough.
- `ugc-ad`: a vertical creator-style ad.
- `social-clip`: a square social clip for sharing.

Each sample can be opened with `@videocontrol` after the local plugin is installed.

Each sample also includes:

- `.videocontrol/intent/project-intent.md`
- `.videocontrol/intent/project-intent.json`
- `.videocontrol/review.json`

Use those files to test whether Codex keeps the same creative direction across later edits.
